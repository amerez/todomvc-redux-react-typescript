## Getting Started

Requirement:

- NodeJS 0.10+


Install dependencies:

```
npm install
```

---

## Running development server

Run webpack dev server (for assets):

```
npm run dev-server
```

Run server:

```
npm run start-dev
```

### Testing

To run tests, use:

```
npm test
```

To start a watch test task:

```
npm run test-watch
```
